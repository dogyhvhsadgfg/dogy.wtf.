# Dogy's Portal

A premium, animated personal landing page built with modern web technologies.

## Features
- **Dynamic Background**: Interactive particle effects with themed glow.
- **Micro-animations**: Smooth transitions, floating widgets, and hover effects.
- **Glassmorphism Design**: High-end visual style using sophisticated CSS variables.
- **Responsive Layout**: Optimized for both desktop and mobile viewing.
- **Integrated Socials**: Quick links to YouTube and TikTok with custom tooltips.
- **Discord Widget**: One-click copy for sharing your Discord identity.

## Technologies Used
- **Vite**: Next-generation frontend tooling.
- **React**: Modern component-based UI library.
- **TypeScript**: Static typing for enhanced development experience.
- **Framer Motion**: Powerful animation library.
- **Lucide React**: Beautifully simple SVG icons.
- **shadcn/ui**: High-quality UI components.
- **Tailwind CSS**: Utility-first CSS framework.
- **Sonner**: Elegant toast notifications.

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [bun](https://bun.sh/)

### Installation
1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```

### Building for Production
To create a production-ready build:
```sh
npm run build
```

## Testing
- **Unit Tests**: `npm run test`
- **E2E Tests**: Standard Playwright tests can be added in the `tests/` directory.
